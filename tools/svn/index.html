# SVN三大指令

  1. checkout

    检出操作：

    • 连接到svn服务器端
    • 更新服务器端数据到本地
    • 注意：checkout只在第一次连接时操作，以后如果需要更新，使用update命令

  2. commit

    提交本地数据到服务端

  3. update

    将服务器的数据同步到本地
